Protocolos de Aplicación

Flujo: del click al video

  1. Resolución DNSDNS / DoH
    ⏱ 0 ms → 15 ms
    GeoDNS devuelve la IP del CDN PoP más cercano. DoH cifra la consulta sobre HTTPS (port 443).
  2. TLS 1.3 HandshakeTLS 1.3
    ⏱ 15 ms → 80 ms (1-RTT)
    Negocia ALPN h3. ECDHE + AES-256-GCM. Perfect Forward Secrecy activa siempre.
  3. HTTP/3 + QUICHTTP/3
    ⏱ 80 ms → 95 ms
    GET al manifest.mpd. Sin HoL blocking. Múltiples streams UDP independientes sobre una sola conexión.
  4. Auth JWT + DRMHTTPS/JWT
    ⏱ 90 ms → 200 ms
    Verifica suscripción. Emite licencia DRM Widevine L1/L3, FairPlay o PlayReady según dispositivo.
  5. MPEG-DASH ABRDASH/HLS
    ⏱ 200 ms → continuo
    Segmentos de 2–4 s en múltiples representaciones. ABR (BOLA/Pensieve) adapta calidad según buffer y throughput.
  6. SMTP + PushSMTP/FCM
    ⏱ Asíncrono
    Emails transaccionales via SMTPS + DKIM/SPF/DMARC. Notificaciones push via FCM (Android) o APNs (iOS).

05

Medidas de Seguridad Propuestas

Modelo de amenaza DDoS volumétrico · MITM / intercepción · Robo de credenciales · Bypass DRM · Exfiltración de datos · BGP hijacking
🔴 Crítico — Red

Protección Anti-DDoS

  • Anycast distribuye tráfico malicioso en todos los PoPs globales.
  • BGP FlowSpec para rate limiting y blackholing automático por IP.
  • Scrubbing centers con filtros stateful (SYN/UDP/ICMP flooding).
  • Capacidad ×5 del pico en cada PoP para absorber ataques.
  • RTBH en el upstream para descarte temprano.
🔴 Crítico — Transporte

Cifrado Extremo a Extremo

  • TLS 1.3 obligatorio. Deshabilitar TLS ≤1.2 y SSLv3.
  • Solo cipher suites fuertes: AES-256-GCM, ChaCha20-Poly1305.
  • Certificate Pinning en la app cliente (anti-MITM).
  • HSTS max-age ≥ 1 año + preload en navegadores.
  • QUIC: sesión cifrada incluyendo encabezados QUIC.
🟠 Alto — Contenido

DRM y Control de Acceso

  • Multi-DRM: Widevine L1/L3, FairPlay, PlayReady según dispositivo.
  • URL signing HMAC-SHA256, TTL de 30 s por segmento.
  • Watermarking forense invisible para rastrear filtraciones.
  • Geoblocking IP + GPS (móvil) por territorios con licencia.
  • Device fingerprinting anti-stream sharing ilegal.
🟠 Alto — Identidad

Autenticación Robusta

  • OAuth 2.0 + OpenID Connect. JWT TTL 15 min + Refresh Token rotativo.
  • MFA (TOTP/SMS) para cambios críticos de cuenta.
  • Detección anomalías: login desde nuevo país → challenge automático.
  • Rate limiting /login: 5 intentos/min → CAPTCHA → lockout progresivo.
🟡 Medio — DNS/PKI

Seguridad DNS

  • DNSSEC firmado en todas las zonas del dominio.
  • DoH en apps para impedir inspección del ISP.
  • CAA records + Certificate Transparency (CT Logs).
  • RPKI para validar rutas BGP (anti BGP hijacking).
🟡 Medio — Email

Anti-Phishing SMTP

  • SPF + DKIM (Ed25519) + DMARC p=reject.
  • BIMI para logo de la empresa en bandejas compatibles.
  • SMTPS (465) / STARTTLS (587). Sin SMTP plaintext.
🟢 Estándar — Aplicación

WAF y AppSec

  • WAF en CDN: bloquea OWASP Top 10 (SQLi, XSS, CSRF).
  • CSP + SRI (SHA-384) en todos los recursos externos.
  • Pentesting trimestral + bug bounty activo.
  • Zero Trust entre microservicios (allowlist explícita).
🟢 Estándar — Compliance

Privacidad y Legal

  • GDPR / CCPA: datos en reposo AES-256. Derecho de supresión.
  • Logs anonimizados (IPs hasheadas, retención 90 días).
  • Data residency EU. SOC 2 Type II / ISO 27001 anuales.
  • Incident Response: notificación GDPR Art.33 en 72 h.

Conceptos Clave

RTT (Round-Trip Time) Tiempo de ida y vuelta de un paquete. RTT = 2×d_prop + d_trans + d_cola. Con CDN hit ≈ 88 ms; sin CDN ≈ 200 ms.
Throughput efectivo Caudal de datos útiles ≤ capacidad del canal. El cuello de botella siempre es el eslabón más lento (generalmente el último kilómetro).
CDN Infraestructura distribuida que aproxima el contenido al usuario, reduciendo latencia y descargando los servidores de origen.

Protocolos

HTTP/3 / QUIC Tercera versión de HTTP sobre UDP+QUIC, eliminando Head-of-Line blocking. 0-RTT resumption. Migración de conexión entre WiFi y 5G.
BGP Protocolo de enrutamiento exterior entre Sistemas Autónomos (AS). Gestiona el Anycast global y la resiliencia del enrutamiento entre ISPs.
TLS 1.3 Handshake en 1-RTT. Elimina cifrados débiles. Perfect Forward Secrecy siempre activa. Integrado en QUIC para HTTP/3.

Modelo OSI Aplicado

Capa 7 — HTTP/3, DNS, SMTP, DASH, DRM
Capa 5–6 — TLS 1.3, sesión segura, cifrado
Capa 4 — QUIC (UDP), TCP (SMTP)
Capa 3 — IP, BGP, MPLS, RPKI, Anycast
Capa 2 — Ethernet, WiFi 6, 5G NR
Capa 1 — Fibra óptica GPON/DWDM, coaxial
